Pro Vôlei – Match Analysis App 🏐
Overview

Pro Vôlei is a prototype application designed to collect and analyze real-time data from volleyball matches. The app enables users to track key in-game events and transform them into meaningful insights for performance evaluation.

This project was initially developed as part of an academic initiative and later evolved into a private project maintained by members of PET Computação.

Features
📊 Real-time data collection
Track points as they happen during a match
Classify points by type:
Serve (Ace)
Attack
Defense
Opponent error
📈 Automatic data visualization
Generate dynamic charts based on match events
Provide an immediate overview of team performance
📄 PDF report generation
Export match statistics into a structured PDF
Useful for:
Post-match analysis
Coaching insights
Performance tracking over time
Purpose

The goal of this app is to support coaches, analysts, and players by providing fast and structured access to match data, helping improve decision-making during and after games.

Status

This repository contains a prototype version of the application for demonstration purposes.
The full project is currently maintained privately by the development team.

Technologies
Flutter (Dart) – cross-platform app development
Material Design – UI components
pdf (Dart package) – PDF creation and chart rendering
printing – PDF handling support
path_provider – file system access
open_file – opening generated files

Disclaimer

This public version is intended only to showcase the concept and initial implementation. Some features may be incomplete or simplified compared to the private version.
