Pro Vôlei – Match Analysis App
Overview

Pro Vôlei is a prototype mobile application designed to collect and analyze real-time data from volleyball matches. The app enables tracking of key in-game events and transforms them into structured insights for performance evaluation.

This project was initially developed as part of an academic initiative and later evolved into a private project maintained by members of PET Computação.

Features
- Real-time Data Collection
- Track match events as they happen
- Classify points by type: Serve (Ace), Attack, Defense, Opponent error
- Automatic Data Visualization
- Generate dynamic charts based on match events
- Provide real-time overview of team performance
- PDF Report Generation
- Export structured match statistics into PDF format
  
Supports
- Post-match analysis
- Coaching insights
- Long-term performance tracking

Purpose

The goal of this application is to support coaches, analysts, and players by providing fast and structured access to match data, improving decision-making during and after games.

Technologies
- Flutter (Dart) – Cross-platform mobile development
- Material Design – UI components
- pdf (Dart package) – PDF generation and chart rendering
- printing – PDF handling and export
- path_provider – File system access
- open_file – Opening generated reports
  
⚠️ Disclaimer

This repository contains a prototype version of the Pro Vôlei App developed within PET Computação.

It is intended only to showcase the concept and initial implementation. Some features may be incomplete or simplified compared to the private production version.

The current production system is part of a private internal project and is not publicly available.
